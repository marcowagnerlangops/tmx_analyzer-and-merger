# TMX Analyzer and Merger (still ongoing, so use it at your own risk)

TMX Analyzer and Merger is an ongoing Streamlit-based project designed to analyze, clean, merge, and QA TMX (Translation Memory eXchange) files.

## Features

- Upload and analyze one or multiple TMX files
- Detect duplicates and inconsistent translations
- Merge TMX files with different merge strategies
- Run QA checks for placeholders, numbers, punctuation, spacing, and formatting
- Compare segments against a glossary
- Repair malformed text and normalize language codes
- Export cleaned results to XLSX
- Export optimized TMX files
- Visual charts and statistics dashboard

## Use Cases

Ideal for:

- Translators
- Localization Specialists
- Language Leads
- Translation Agencies
- LangOps workflows
